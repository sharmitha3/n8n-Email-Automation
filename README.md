# n8n-Email-Automation

## Objective
To design and implement an automated email workflow using n8n and AI, enabling seamless, personalized email communication with minimal manual effort, improving efficiency and reliability in routine email tasks.

## Project Overview
This project is an AI-powered email automation system built using n8n workflows. It allows users to send emails automatically via a chat interface. For example, you can send personalized birthday emails or routine notifications without manually composing each message.

The workflow integrates:

1.User messages → sent to the AI Agent
2.AI processing → via OpenRouter GPT-Turbo model
3.Memory/logic node → processes the message and decides action
4.Gmail action → sends the email automatically
5.This setup allows you to have a smart, fully automated email system that reacts to chat commands.

## Features

-AI-powered email generation
-Fully automated workflow using n8n
-Gmail integration for sending emails
-Memory node for context-aware actions
-Easy to customize for different email types
